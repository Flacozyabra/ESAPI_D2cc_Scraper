# План реализации: Улучшение визуального стиля и добавление лога сплиттера

Этот план описывает улучшения интерфейса PyQt6 приложения ESAPI D2cc Scraper: замену прочерков на `n/a`, улучшение визуала главного окна и настроек, скрытие критических ошибок при старте и внедрение панели лога с кнопкой-стрелочкой для сворачивания.

## Список изменений по файлам

### 1. [core/esapi_worker.py](file:///c:/Users/Falco/Desktop/ESAPI%20D2cc%20Scraper/core/esapi_worker.py) [MODIFY]
- Заменить все прочерки `-` на `n/a` в словаре результатов (в полях `"sd"` и `"td"`).

### 2. [gui/main_window.py](file:///c:/Users/Falco/Desktop/ESAPI%20D2cc%20Scraper/gui/main_window.py) [MODIFY]
- **Замена прочерков**: при сбросе данных в метках выводить `n/a` вместо `-`.
- **Подавление ошибок при старте**: при неудачном первичном подключении к ESAPI не вызывать блокирующий `QMessageBox.critical`, а просто выводить ошибку в лог и статус-бар.
- **Поле лога снизу**:
  - Добавить `QTextEdit` для вывода логов (ошибки, предупреждения, статус подключения).
  - Сделать лог только для чтения (ReadOnly) с темным фоном.
  - Разместить над логом разделительную плашку (тонкий сплиттер) с широкой плоской кнопкой по центру (стрелочка ▲/▼).
  - Реализовать функцию `toggle_log()`, которая скрывает/показывает лог и меняет стрелочку.
- **Улучшение визуала**:
  - Настроить скругление углов главного окна (добавить `border-radius: 8px` в QSS темы и включить `TranslucentBackground` для окна, если необходимо).
  - Добавить тонкую контрастную границу окна (`border: 1px solid #3e3e42`), имитирующую тень в безрамочном режиме.

### 3. [gui/settings_window.py](file:///c:/Users/Falco/Desktop/ESAPI%20D2cc%20Scraper/gui/settings_window.py) [MODIFY]
- **Исправление багов с рамкой ОС**:
  - Изменить флаги окна настроек на `Qt.WindowType.CustomizeWindowHint | Qt.WindowType.FramelessWindowHint`, чтобы ОС Windows гарантированно не рисовала светлый системный заголовок поверх кастомного темного.
- **Улучшение визуала**:
  - Сделать рамку и скругление углов такими же, как у главного окна.

### 4. [gui/themes/dark.py](file:///c:/Users/Falco/Desktop/ESAPI%20D2cc%20Scraper/gui/themes/dark.py) [MODIFY]
- Обновить общие стили QSS: добавить скругления для `QMainWindow`, `QDialog`, `QLineEdit`, `QComboBox` и кнопок.
- Добавить стилизацию панели сплиттера и кнопки лога.
